# 📖 Project Overview

**MediQueue** is a modern tutor booking platform designed to connect students with verified tutors around the world.

Students can browse tutor profiles, explore tutor details, book sessions, and manage their booked lessons effortlessly.

Tutors can create professional tutoring profiles and provide personalized learning experiences for students.

The platform focuses on:

- Premium UI/UX
- Secure Authentication
- Real-Time Tutor Discovery
- Session Booking Management
- Responsive Design
- Smooth User Experience
